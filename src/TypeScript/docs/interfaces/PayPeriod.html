  • Get the calendar pattern number used for this pay-period. It is a value between 1 and 14 (inclusive).

  • get startDay(): number

  • Get the day of the month that the pay-period starts on. See startMonth

startMonth

  • get startMonth(): number

  • Get the month that the pay-period starts in; it may be different than the pay-period month. See startDay

    Returns number

workDays

  • get workDays(): number

  • Get the number of work days in the pay-period. It is either 21 or 22. The number of work days is based on a standard full-time schedule from Monday thru Friday.

    Returns number

workHours

  • get workHours(): number

  • Get the number of work hours in the pay-period. This is simply the workDays × 8.0 and is either 168 or 176 hours.

    Returns number
