ca-gov-payperiods

CA Gov Pay Periods API in TypeScript (JavaScript)

A Pay Periods API for standard monthly pay periods for California State Government as defined in the State Administrative Manual section 8500.

Standardized montly calendars are defined from 1994 thru 2299. The repeating pattern fails in 2300.
