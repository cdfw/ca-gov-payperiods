    Function ical

    • ical(payPeriods: PayPeriod[], name?: string): string

      Given an array of pay periods, return a string formatted as an iCalendar message with the pay periods as events.

      Parameters

      • payPeriods: PayPeriod[]

        One or more payperiods to write as events in the calendar.

      • Optionalname: string

        (default: "Pay Periods") Optional name for the calendar.

      Returns string

      The pay periods formatted into an iCalendar message.

      Example

      // Get the pay periods for a year.
      const payPeriods = getPayPeriods(2020);
      // Get the iCalendar representation of that year.
      const msg = ical(payPeriods, "2020 Pay Periods");

      BEGIN:VCALENDAR
VERSION:2.0
PRODID:CA-GOV-PAYPERIOD
X-WR-CALNAME:2020 Pay Periods
BEGIN:VEVENT
UID:CA-GOV-PAYPERIOD-2020-01
DTSTAMP:20200731T190554Z
DTSTART:20200101
DTEND:20200131
SUMMARY:January 2020
DESCRIPTION:January 2020 Pay Period
 \nFirst Day: 2020-01-01
 \nLast Day: 2020-01-30
 \nWork Days: 22
 \nWork Hours: 176
END:VEVENT
...
END:VCALENDAR
